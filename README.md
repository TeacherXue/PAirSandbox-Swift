# PAirSandbox-Swift
PAirSandbox Swift版  仿照MrPeak的PAirSandbox  可方便实时查看沙盒中的文件并传送给mac

使用方法:
  将PAirSandbox.swift拖入工程，在AppDelegate初始化即可，然后在应用中从右往左划即可呼出
